# Supplementary Audio Samples

This repository provides supplementary audio samples for the paper:

**Drone Audition: Lifting-Based U-Net for Intelligibility-Preserving Multi-Channel Speech Enhancement**

Yifei Wei, Jihui (Aimee) Zhang, Benjamin Yen, and Yusuke Hioka  
19th International Workshop on Acoustic Signal Enhancement (IWAENC 2026)

## Audio Samples

The audio files correspond to the spectrogram examples presented in the paper at an average input SNR of −25 dB for microphone channel 2.

| No. | File                              | Description                     |
| --- | --------------------------------- | ------------------------------- |
| 1   | `Figure3_(a)_Clean_reference.wav` | Clean reference speech          |
| 2   | `Figure3_(b)_UAV_ego-noise.wav`   | UAV ego-noise                   |
| 3   | `Figure3_(c)_Mixture.wav`         | Noisy speech mixture            |
| 4   | `Figure3_(d)_M1.wav`              | Output of the proposed M1 model |
| 5   | `Figure3_(e)_M2.wav`              | Output of the proposed M2 model |
| 6   | `Figure3_(f)_Wave-U-Net.wav`      | Output of Wave-U-Net            |
| 7   | `Figure3_(g)_MC-Conv-TasNet.wav`  | Output of MC-Conv-TasNet        |
| 8   | `Figure3_(h)_DWT-Wave-U-Net.wav`  | Output of DWT-Wave-U-Net        |
| 9   | `Figure3_(i)_MWF-GMM.wav`         | Output of MWF-GMM               |
